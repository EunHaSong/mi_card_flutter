# mi_card_flutter
자기소개 카드 앱을 만드는 실습 예제입니다.

## 설치 방법
```bash
git clone https://github.com/woojin-devv/mi_card_flutter.git
```
혹은 zip 파일 다운로드 후 압축 해제

### 로컬 디렉토리 구조
```
.
├── members/                   
│   └── woojin/                  
│       ├── WEEK01/               
│       │   ├── 1주차정리.md      
│       │   └── practice/        
│       └── WEEK02/               
│           ├── 2주차정리.md
│           └── practice/     # 👈 여기에 클론할 project 루트가 위치하도록 해주세요
├── README.md                    
```
## 참고사항 
- `example` 브랜치에서 예제 코드를 확인하실 수 있습니다.
